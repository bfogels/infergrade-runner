# InferGrade Runner 0.3.33

Runner 0.3.33 advances the contract to `0.3.22` and unlocks two exact
current-generation Assistant calibration targets after artifact, fit, runtime,
and output-protocol canaries.

## Changed

- Qwen3.5 4B Q4_K_M is now a reviewed-runnable Apple Silicon campaign target.
- Ministral 3 8B Q4_K_M is now a reviewed-runnable Apple Silicon campaign
  target.
- Gemma 4 12B and Qwen3.6 remain blocked behind their separate artifact,
  runtime, fit, hardware, and protocol gates.

## Evidence boundary

This release changes which exact setups may be selected for evidence
collection. It does not change Capability protocol v3.1 tasks, weights, raw
attainment, calibration thresholds, or published scores. Hub must still pin
the exact immutable artifacts and this release bundle before scheduling either
target, and the resulting runs must still prove their own capability,
performance, memory, and repeatability claims.
