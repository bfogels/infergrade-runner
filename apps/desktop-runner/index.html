Step 1

Pair with Hub

Use HTTPS for hosted Hubs. Local development can use localhost or 127.x.x.x.

Click Pair Local Runner in the Hub, then paste the one-time code here.

Pairing stores the runner token in the OS credential store. Tokens are not shown in this browser UI.

Steps 2 and 3

Install runtime and choose a model

Use the recommended llama.cpp runtime and a local GGUF file for the first benchmark.

The Hub owns account and token state. This app keeps the local runtime, model file, and listener on your Mac.

Run first local benchmark

Start from a Hub run handoff when available. For a local smoke test, download the starter model and run a short native benchmark.
